# Unit Converter (Python GUI)

A simple GUI-based unit converter built using Python and Tkinter.

## Features
- Convert KM ↔ Miles
- Convert °C ↔ °F
- Convert KG ↔ LBS
- User-friendly interface

## Technologies
- Python
- Tkinter
- PIL (Pillow)

## How to Run
1. Install dependencies:
   pip install pillow
2. Run:
   python main.py
